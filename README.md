# MockMate AI

MockMate AI is an AI-powered mock interview platform that helps users practice technical interviews with real-time interaction, speech recognition, coding rounds, resume-based question generation, and AI-driven feedback.

---

# Features

- AI-powered mock interviews
- Speech-to-text answer recording
- Resume upload and analysis
- Coding interview environment
- Real-time interview questions
- AI-generated feedback system
- User authentication and profile management
- Responsive modern UI
- Protected routes and dashboard

---
